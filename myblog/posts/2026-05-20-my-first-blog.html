🛠 博客搭建过程

  1. 创建 GitHub 仓库 blog
  2. 上传 HTML 和 CSS 文件
  3. 配置首页 index.html 和文章列表
  4. 添加第一篇文章 Markdown 文件到 posts/
  5. 链接文章 到首页，确保点击查看文章详细内容

💡 学到的经验

  • 使用 Markdown 编写文章，方便维护和阅读
  • 保持文章文件命名规范，例如 YYYY-MM-DD-标题.md
  • 每篇文章都应该有 Front Matter（title、date、category、description）
